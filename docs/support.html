Support — PNG to Slides

How to Use

  1. Open a presentation in Canva
  2. Select the page after which you want to insert slides
  3. Open the PNG to Slides app from the sidebar
  4. Click the file picker and select your PNG files (you can select multiple)
  5. Click Insert Slides
  6. Wait for the progress bar to complete

Frequently Asked Questions

What file formats are supported?

Only PNG files (.png) are supported. Other image formats (JPEG, SVG, etc.) will be rejected by the file picker.

What order are the slides inserted in?

Files are sorted alphabetically by filename. Use zero-padded names for correct ordering:

Good: page_001.png, page_002.png, ..., page_099.png
Bad: page_1.png, page_2.png, ..., page_10.png (10 sorts before 2)

Where are slides inserted?

New slides are added immediately after the currently selected page in your presentation. Select the page you want to insert after before clicking "Insert Slides".

How are images sized?

Each image is scaled so its height matches the page height, and it's centered horizontally. If the scaled image is wider than the page, the sides will extend beyond the edges.

Is there a file size limit?

Individual PNG files should be under 10 MB. Files are processed as data URLs in the browser, so very large files may cause performance issues.

Why does it say "Adding pages is not supported"?

The addPage() API is only available in certain Canva design types (like Presentations). It's not supported in Whiteboards, Docs, or some other design types.

The app seems slow with many files?

The Canva API limits page additions to 3 per second. The app waits 350ms between each page insertion to stay within this limit. For 100 slides, expect about 35 seconds.
